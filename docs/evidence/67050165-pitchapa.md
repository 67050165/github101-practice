# GitHub Practice Evidence — [พิชชาภา คงมหาพฤกษ์]

## What I did
อัปเดตไฟล์ข้อกำหนดผลิตภัณฑ์ (README.md), อัปโหลดไฟล์โมเดล 3 มิติหัวฉีดพ่นโฟม (.STL) และภาพหน้าจอหลักฐานชิ้นงานลงในคลังข้อมูลของทีม

## My role in project
Product / UX Designer

## Evidence link
ไฟล์ทั้งหมดจัดเก็บอยู่ในโฟลเดอร์ ux-product ในคลังข้อมูลกลาง (Repository) ของทีม พร้อมลิงก์โครงการสาธารณะบน Tinkercad เพื่อใช้ตรวจสอบมิติชิ้นงาน

## Reflection
เรียนรู้การจัดโครงสร้างไฟล์งานออกแบบ (Design Artifacts) ให้เป็นระบบ, การใช้ภาษา Markdown ในการเขียนรายงาน และระบบตรวจสอบเวอร์ชัน (Version Control) เพื่อส่งมอบไฟล์โมเดล 3D ให้เพื่อนในทีมนำไปใช้งานต่อได้ทันที

ฉันติดปัญหาอะไร:สายยางจากปั๊มน้ำยาโฟมหลุดง่าย เนื่องจากท่อเข้า (Inlet) ของหัวฉีดเดิมเรียบเกินไปและไม่มีแรงบิดล็อก

ฉันแก้ปัญหาอย่างไร: ออกแบบขนาดท่อเข้าให้ยาว 15 mm และขยายขนาดภายนอกเป็น 8 mm พอดีกับสายยางมาตรฐาน เพื่อสร้างแรงเสียดทานให้เสียบแน่นหนาและไม่หลุดขณะมีแรงดันพ่นออกมา
